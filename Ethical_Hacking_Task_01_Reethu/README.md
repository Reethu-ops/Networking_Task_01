# Ethical Hacking Task 01 – Information Gathering & Reconnaissance

## Overview

This repository contains the work completed for **Ethical Hacking Task 01**, which focuses on the reconnaissance (information gathering) phase of ethical hacking. The objective of this task is to collect publicly available information about a target website using passive reconnaissance techniques without interacting with or exploiting the target system.

## Target Information

* **Website Name:** OpenAI
* **Target URL:** https://openai.com

## Objectives

* Perform passive information gathering.
* Conduct a WHOIS lookup.
* Identify DNS records.
* Analyze website technologies.
* Check HTTP security headers.
* Examine the robots.txt and sitemap.xml files.
* Document all findings with screenshots and a detailed report.

## Tools Used

* WHOIS Lookup Service
* WhatsMyDNS
* BuiltWith / Wappalyzer
* SecurityHeaders.com
* Web Browser
* Visual Studio Code
* GitHub

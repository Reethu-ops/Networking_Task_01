# Linux Task 03 – Process & System Monitoring

## Objective

To understand Linux process management, system monitoring, service monitoring, security monitoring, and basic shell scripting.

## Environment

* Operating System: Kali Linux
* Platform: VirtualBox (Virtual Machine)

## Topics Covered

* Process Monitoring
* Process Management
* System Monitoring
* Service Monitoring
* Shell Scripting
* Security Monitoring

## Activities Performed

* Monitored running processes using ps, ps aux, top, and htop.
* Identified process IDs (PID) and checked CPU and memory usage.
* Created a test process using sleep 300.
* Located the process using ps aux | grep sleep.
* Terminated processes using kill and kill -9.
* Collected system information using free -h, df -h, uptime, and uname -a.
* Checked service status using systemctl status.
* Created a shell script system_report.sh for system information reporting.
* Performed basic system and security monitoring tasks.

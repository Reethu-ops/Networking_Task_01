# Linux Task 01

## Objective

To learn basic Linux terminal operations, file management, directory navigation, and system information commands using Kali Linux.

## Environment

* Operating System: Kali Linux
* Platform: VirtualBox

## Activities Performed

* Practiced Linux terminal commands
* Created and managed directories
* Created, copied, moved, renamed, and deleted files
* Collected system information
* Explored Linux file structure

## Commands Used

* pwd
* ls
* ls -la
* cd
* clear
* history
* whoami
* hostname
* mkdir
* touch
* cp
* mv
* rm
* tree
* uname -a
* date
* uptime

## Skills Gained

* Linux terminal navigation
* File and directory management
* System information gathering
* Basic command-line operations
* Understanding Linux file structure

## Conclusion

This task provided practical experience with Linux commands, file management, and system administration basics. It strengthened foundational Linux skills that are important for cybersecurity and ethical hacking.

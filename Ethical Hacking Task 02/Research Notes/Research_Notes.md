# Research Notes

## Task

Ethical Hacking Task 02 – Nmap Network Scanning

## Objective

To understand how Nmap is used to identify open ports, running services, service versions, and operating system details.

## Topics Studied

* Nmap Installation
* TCP Port Scanning
* Service Version Detection
* Operating System Detection
* Common Network Ports
* Network Reconnaissance

## Commands Used

```bash
nmap --version
nmap localhost
nmap -sV localhost
nmap -O localhost
```

## Tools Used

* Nmap
* Command Prompt
* Visual Studio Code
* GitHub

## Learning Outcome

This task improved my understanding of network scanning, port enumeration, service identification, and operating system fingerprinting. I learned how security professionals use Nmap during penetration testing while ensuring that scans are performed ethically on authorized systems only.

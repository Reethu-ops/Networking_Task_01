# Ethical Hacking Task 02 – Nmap Network Scanning

## Overview

This repository contains the work completed for Ethical Hacking Task 02. The task demonstrates how to use Nmap for network reconnaissance by scanning the local machine to identify open ports, detect running services and their versions, perform operating system detection, and understand common network ports.

## Objectives

* Install and configure Nmap.
* Perform a local machine scan.
* Detect running service versions.
* Detect the operating system.
* Study common TCP ports.
* Analyze scan results.
* Document findings.

## Tools Used

* Nmap
* Command Prompt
* Visual Studio Code
* GitHub

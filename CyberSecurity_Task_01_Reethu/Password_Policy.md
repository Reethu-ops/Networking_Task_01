# Password Security Review

## What makes a strong password?

A strong password contains uppercase letters, lowercase letters, numbers, and special characters. It should be at least 12 characters long.

## Why should passwords be unique?

Unique passwords prevent attackers from accessing multiple accounts if one password is compromised.

## What is MFA?

Multi-Factor Authentication (MFA) requires users to provide more than one method of verification before accessing an account.

## Risks of Password Reuse

If one account is hacked, attackers can use the same password on other accounts.

## Password Policy

1. Use at least 12 characters.
2. Include uppercase and lowercase letters.
3. Include numbers and special characters.
4. Avoid personal information.
5. Enable MFA whenever possible.

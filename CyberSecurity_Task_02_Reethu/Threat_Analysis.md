# Threat Analysis

## Insider Threat

Description:
A trusted person misuses access privileges.

Impact:
Data theft and unauthorized changes.

Prevention:
Access control and monitoring.

---

## Malware

Description:
Malicious software that damages systems.

Impact:
Data loss and system disruption.

Prevention:
Antivirus and software updates.

---

## Phishing

Description:
Fake emails or messages used to steal information.

Impact:
Credential theft and financial loss.

Prevention:
User awareness and MFA.

---

## Weak Passwords

Description:
Passwords that are easy to guess.

Impact:
Unauthorized account access.

Prevention:
Strong password policies.

---

## Unpatched Systems

Description:
Systems missing security updates.

Impact:
Exploitation of vulnerabilities.

Prevention:
Regular patch management.
